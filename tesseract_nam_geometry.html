NAM Geometric Addressing
Neural Addressable Memory — Technical Overview
The foundation
Information navigated by address,
not ranked by similarity.
NAM assigns every piece of information a deterministic coordinate in n-dimensional semantic space. Retrieval navigates directly to that address. The same query always produces the same coordinates, always reaches the same records. No probability. No ranking. No hallucination. If nothing exists at the address, the system says so.
NAM vs vector search
NAM — deterministic
Navigates by address. Computes the target coordinate directly from the query.
Same query, same result. Every time, every environment, every run.
Rejection is first-class. Nothing at this address is a structural answer, not a failure.
Fully auditable. Every retrieval decision is replayable and traceable.
Zero cloud dependencies. Runs entirely inside the fence.
Vector search — probabilistic
Ranks by similarity. Scans the space and scores every candidate.
Non-deterministic. Same query returns different results depending on state and load.
No rejection primitive. Always returns something — including wrong answers.
Not auditable. Retrieval decisions cannot be explained or replayed.
Cloud-dependent. Requires external APIs and connectivity.
